# **App Name**: Prompt Engineering Trainer

## Core Features:

- Scenario-Based Prompt Generation: Generates a structured prompt based on selected scenario, guiding users to define roles, tasks, and constraints for effective prompt engineering.
- AI-Powered Prompt Review: Leverages AI to evaluate the generated prompt against predefined criteria (clarity, context, format, structure) and provides detailed feedback with suggestions for improvement.
- Mini-Test Evaluation: The LLM will act as a tool to evaluate the user's answers to questions based on information from the training content and provide relevant feedback to the user.
- AI Chat Assistant: Offers a real-time AI chat assistant for users to ask questions and receive guidance on prompt engineering concepts and best practices.
- Prompt Examples: View well-formed prompt examples.

## Style Guidelines:

- Primary color: Forest green (#38A3A5), evoking growth, learning, and a tech-forward approach.
- Background color: Very light grayish cyan (#F0FAFA), offering a clean and calm backdrop to focus on learning.
- Accent color: Muted blue (#577590), providing contrast and highlighting key interactive elements without overwhelming the user.
- Body and headline font: 'PT Sans' for a balance of approachability and modernity.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent, clear, and professional icons from the Material Design icon set to represent different features and concepts.
- Implement a clean and structured layout with clear section headings and generous spacing to improve readability and guide users through the learning process.
- Employ subtle animations and transitions to provide feedback and enhance user engagement, avoiding distracting or unnecessary effects.
